#### ⚙️ Система модификаторов

**Создана новая модульная система модификаторов**
- `module/core/modifier-system.js` (новый файл) — централизованная система обработки модификаторов
  - Категоризация модификаторов: атрибуты, броски атрибутов, броски атаки, защитные броски, общие модификаторы, характеристики, защита, эффекты со временем
  - Нормализация и агрегация модификаторов
  - Поддержка плоских (flat) и множительных (mult) модификаторов
  - Система лейблов для отображения в UI
  - Диалог редактирования модификаторов с группировкой по категориям

- `module/core/modifier-registry.js` (новый файл) — реестр всех модификаторов
  - 69 типов модификаторов с параметрами stack (способ суммирования) и default (значение по умолчанию)
  - Поддержка advantage/disadvantage/lucky/unlucky/dice для всех атрибутов
  - Поддержка overtime-эффектов (DoT/HoT)

**Рефакторинг системы эффектов**
- `module/effects.js` — полная переработка
  - Переименование `normalizeEffects` → `normalizeModifiers`
  - Разделение эффектов advantage/disadvantage на отдельные модификаторы (adv/dis/lucky/unlucky)
  - Удаление legacy-кода и функции `openEffectsDialog`
  - Интеграция с новой системой `modifier-system.js`
  - Упрощение сбора эффектов: `collectEffectTotals` теперь делегирует к `computeActorTotals`
